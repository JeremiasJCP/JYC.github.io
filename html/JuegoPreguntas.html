Cómo jugar

  1. El juego se juega con dos jugadores.
  2. Cada jugador toma un turno para presionar el botón "Jugar".
  3. Cuando se presiona el botón, se genera una tarjeta con una categoría y una palabra o acción aleatoria.
  4. El jugador debe entonces completar el desafío que se le indica en la tarjeta y el otro jugador debera darle indicaciones adicionales para que se pueda completar la ronda.
  5. La finalidad es divertirnos mientras hacemos algo distinto
